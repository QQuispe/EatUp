3. Summary of group project, accomplishments and learning experience

EatUp project is an interactive web application to find the nearest restaurant between two people. The application provides users with two auto-fill in address bar and option to choose their flavor and it automatically outputs the nearest and the best restaurant which fits to their flavor. We leverage Django Web frame work in this project to develop a very user-friendly application. In addition we have applied various APIs such as Yelp, Google Place, and Google Geocoding to come up with the best restaurant for the users. All the backend support and interactions with frontend developed using Python language.

  • Developed an interactive web application using Django web framework
  • Completed Front-End using HTML
  • Completed Back-end using Python language
  • Worked with Google Place API
  • Worked with Geocoding API
  • Worked with Yelp API

Colin:

My portion focused on the google geocaching API and the usage of geographic coordinates. It was my job to ensure that when a user enters two addresses, that the restaurant(s) closest to both locations are identified. I also developed a visualization of data out of the Yelp API.

Google Geocaching API – To find the restaurant closest to two locations I first needed to get the coordinates of each address. I send each address through the API returning a latitude and longitude for each.

JSON – The API returns JSON data with the latitude and longitude values. I parse the JSON to extract the values.

Dictionaries, Lists and Functions - The coordinates are passed to a simple function that find the average or each. The average coordinates is the center point.

Pandas and Matplotlib – I use JSON data from the Yelp API Farhan developed to create data visualization for the results page.

Other libraries – pprint library improve the readability and formatting when printing JSON data. I also used the plotly library to assist with visualization.
